# Spotifme
This app was created to display user Spotify playlists.

### FrontEnd
* JavaScript,
* React.JS,
* Spotify Api,
* Spotify Oauth


### Backend
* Node.JS
* Heroku

### Deployed to Heroku
https://spotifme.herokuapp.com
